1.阅读并运行本目录下的netbeans项目文件夹demoGraph中的代码，写出各个类中方法之间的调用关系。

答：

主类的引入语句：
package TestMenu;
import MyGraph.Circle;
import static javax.swing.JOptionPane.*;
import static MyGraph.MyMenu.*;
import MyGraph.MyShape;
import MyGraph.Rectangle;
import MyGraph.Triangle;

主类和其他调用方法的类不在同一个包中。它通过静态导入JOptionPane类中的所有静态方法、MyGraph中的MyMenu类以及的其中所有静态成员方法（import static），使之可以在主类中可以直接调用这些方法，而不用通过建立对象再进行调用方法。
对于其他非静态导入的包中的类(import)，就需要通过调用对象来调用相关成员方法。
